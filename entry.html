🔑 GitHub-token

Nodig om rechtstreeks naar de site te publiceren. Eenmalig aanmaken: GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token → Repository access: alleen TDF2026 → Permissions → Contents: Read and write. Het token wordt alleen lokaal in je browser bewaard.

📋 Etappe-uitslag invoeren

Kies de etappe en vul de top 10 in met rugnummers (positie 1 = winnaar). Bestaat er al een uitslag voor deze etappe, dan wordt die automatisch getoond zodat je 'm kunt aanpassen.

💾 Alternatief: handmatig opslaan

Publiceren via token is niet gelukt. Download het bestand hieronder en plaats het zelf als data/results.json in de repo.
